Welcome to **My Web Hub**, a central portfolio of my web-based projects. This repository serves as a sandbox for experimentation and a showcase of my skills in frontend development, JavaScript logic, and mathematical problem-solving.

## Projects

### Assignment Tracker  
A productivity tool that calculates how many assignments must be completed each day to stay on track across multiple classes.

**Demonstrates:**  
- User input validation  
- Stateful calculations across timeframes  
- Practical problem-solving with JavaScript  

---

### Snake Game  
A classic Snake implementation featuring power-ups and obstacles.

**Demonstrates:**  
- Game loops and state management  
- Collision detection  
- Dynamic difficulty and game logic  

---

### Social Hub (Link Tree)  
A lightweight link-tree style page used to organize and present my online presence and projects.

**Demonstrates:**  
- Clean UI design  
- Responsive layout  
- Simple navigation and presentation

### Game Hub

#### Blackjack  
A fully playable Blackjack game included to demonstrate my ability to work with existing codebases, APIs, and third-party libraries.

**Demonstrates:**  
- Reading, understanding, and extending external code  
- Proper attribution and citation of third-party resources  
- Integration of libraries such as jQuery and Bootstrap  
- Managing complex game state and user interactions  

---

#### Aim Trainer  
A browser-based aim and reaction trainer implemented using **pure vector math** rather than DOM-based shortcuts.

**Demonstrates:**  
- 2D vector math for position, movement, and direction  
- Distance-based hit detection  
- Time-based updates independent of frame rate  
- Mouse input handling and real-time interaction
